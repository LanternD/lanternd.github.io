# 　

## 致敬老员工

![img]({{site.img-hosting}}/Pic4Post/launching-new-mac-studio/old-mac-book-pro-corner.jpg)
*摔了一个角的旧 MacBook Pro*

　2019 年买的 Intel 的 MacBook Pro 服役至 2025 年显然已经超龄。

　它出现了电池问题。表现为如果不插电源，一段时间后开机就会提示电量不足；插上电源半分钟后才能开机。开机了又会发现电量其实还过半。另外还有鼓包，底部的四个点都有点不着地了，整个笔记本仿佛个陀螺，可以旋转。可能有点夸张了，总之就是电池老化了。

　电池问题可能换一下即可复活。但是总体来说还是性能赶不上了。就视频处理来说，2019 年的时候处理 4K 30帧视频就撑死了，录制规格也都是 H.264，设备也没现在这么丰富。我平时录了很多视频。虽说不一定会剪，但是看到这旧 Mac 的卡顿的界面，那就是一点也不想剪了。于是素材越积越多（给懒惰找了充足的借口）。

　在视频处理以外，3D 建模、图片处理、软件编译等等都存在着不流畅的使用体验。

　再者，500 GB 的硬盘更是让我捉襟见肘。有时候想升级一下系统都得花点时间看看哪些东西还能处理一下放到云上或者 NAS 才能腾出地方下载升级包。

　最后就是发热问题。放 2019 年的话，这点热量也已经够低了，全靠同行衬托。但是大家都知道，M 系列芯片出现以后，电脑发热就被一脚踹进了历史的垃圾堆。想让 M 系列电脑风扇转起来都得努力想点办法。而我的 MBP 放腿上用一段时间都有种低温烫伤的趋势，发热量不容小觑。

　老实说这些问题又都是忍一忍也能用下去的。但 Anyway，只要出现了要换电脑的想法，那就看哪里都是缺点。

　当然，我对这位 Intel MacBook Pro 老员工还是充满了敬佩。毕竟它陪我走过了求学和工作的漫漫长路。家里的 3D 打印建模、DIY 的电路设计，各种小型网页和桌面应用等等也都出自它之手。

　「神龟虽寿，犹有竟时；螣蛇乘雾，终为土灰」。在偶尔有移动需求的时候它应该还能派上用场，但是现在历史也该翻篇了。

## 欢迎新员工入职

![img]({{site.img-hosting}}/Pic4Post/launching-new-mac-studio/new-mac-studio-edge.jpg)
*新 Mac Studio 的边缘折痕，非常干脆俐落*

　上个月买了台新的电脑 —— Mac Studio。大概配置如下：

- Apple M4 Max
	- 14 核 CPU，32 核 GPU，16 核 Neural Engine
- 36 GB RAM
- 2 TB 存储

　可以说就是大容量的「丐版」（意思是最低的配置）。为什么要买丐版呢？因为已经完全够我用了。按我的理解，Mac Studio 粗浅来分类的话，要么就是够用的入门配置，要么就得把配置拉满，跑 LLM 大模型、各种天花乱坠的渲染和剪辑。盲猜可能很少人买中间的配置吧？目前我还不想上那么多钱加内存跑那些本地大模型，于是就按奥卡姆剃刀原则买了「够用」版本。

　另外我还有个 iPad 来负责大部分「移动」需求，所以这次我没有再选 MacBook Pro。毕竟 MBP 的屏幕素质是相当给力的，所以目前代价就是，我得用色彩没这么「正」的第三方显示器来负责图片和视频的颜色处理了。

　我确实在高配 / 顶配 Mac Mini 和这个丐版 Mac Studio 之间纠结了一下。大概就是 M4 Pro 和 M4 Max 的区别。M4 Pro 的性能也基本够我用了。我想着上一个 Intel 笔记本都能用超过 6 年，那么加点钱上 Max 就能多撑一段时间（晚点被淘汰）也是划算的。而且在这很长的时间里我都能享受那多出来的一点性能盈余，何乐而不为？

## 初始化

　我在过去很长一段时间里干过多次「初始化 Linux」的事情，包括各种服务器、虚拟机，以至于我写过个「[Ubuntu（或者 Debian）装机笔记](https://en.dlyang.me/things-to-do-after-installing-ubuntu/)」（很老的文章了）给自己参考。而初始化 MacOS 却是我很少干的事情。除了公司换电脑的时候稍微折腾过（其实也只是数据传输一下），我自己从来没有重装或初始化过一台 MacOS 的机器。上一台 Intel MBP 有太多历史遗留问题。我这次决定「重新做人」—— 设定为新的 Mac。

　这里写写第一次 Mac 开机以后要干的事情（大致按时间排序但不绝对），给未来的自己参考一下：

- 把默认 Dock 给清理一下，不常用的 App 移除，同时添加几个 Dock 必要的自带 App，比如 Dictionary
- 配置同步盘和 Joplin，同时挂机让它们完成同步
- 改 hostname，取个响亮的名字
- 配置输入法：添加中文和小鹤双拼
- 系统快捷键：
	- 修改 Modifier Key，把 <kbd>Caps Lock</kbd> 映射到 <kbd>Command ⌘</kbd>
	- LaunchPad 快捷键：<kbd>Option⌥ + x</kbd>
	- Mission Control：<kbd>Option⌥ + Tab</kbd>
- 在浏览器登陆各种账号
- 下载 Karabiner-Elements 并实现：
	- 单击 Shift 键切换输入法（默认是 <kbd>Control ⌃ + Space</kbd>，太麻烦）
	- <kbd>Cmd ⌘ + F1 - F4</kbd> 分别打开 Finder、Safari 浏览器、Warp（终端）、Message
- 下载 Rectangle：方便调整窗口位置和大小
- 配置各种快捷指令（Quick Action）：
	- [Pangu](https://github.com/vinta/pangu.js)：自动格式化文字，给字母和汉字之间加个空格
	- 新建 Markdown 文档，并绑定到 <kbd>Cmd ⌘ + Shift ⇧ + M</kbd>
	- 获取图片或 STL 文件的路径，格式化以后放到剪贴板：用来写博客的
- 添加各种邮箱账号，并配置日历📅、笔记等
- 下载日常软件：
	- 即时通讯软件：WeChat、电报
	- 看视频的：IINA
- 下载生产力工具
	- AI：ChatGPT 的 Mac 应用
	- 思维导图：XMind
	- 软件开发：Brew（用 `brew` 安装的就不写了，太杂）、Zed、VSCode（嵌入式开发）、Cursor、Warp
	- EDA 软件：FreeCAD、KiCAD
	- 剪视频的：Final Cut Pro、360 Fusion Studio
	- 处理照片的：Adobe LightRoom Classic、Photoshop
- 装一些 Safari 浏览器插件，比如 VimLike 和去广告的
- 配置写博客的环境：Jekyll / Bundler、Ruby、AWS、GitHub repo 等等
- 配置 SSH Key 给各种代码库（GitHub 等）

　我曾以为东西很多，其实两个晚上就弄得差不多了。

## 目前体验

　外观：Mac Studio 真机要比想象中大点。和台式机机箱比起来却已足够袖珍。性能比它强的体积都得比它大；体积和它一样的性能都差了老远。包装盒里面除了机器就是一根电源线，相当简洁了。它顶上是一个很不错的平面，是不可多得的储物空间。我经常在上面临时摞多余的笔记本或者 iPad。

　接口：我的显示器可以用 USB-C 供电加数据传输，所以目前没有用 HDMI 口。那一排 USB 接口我用起来也是绰绰有余了。网线也没插，因为 1️⃣ 房间离路由有点远，2️⃣ WiFi 网速也够用。

　性能：我在公司还是用过 M4 Max 芯片的电脑的，所以这个方面并没有从 Intel 直接跳到 M 系列芯片的惊喜。不过我自己常用的 App 打开都无比丝滑，告别了以前的卡顿。舒服了。网上测评 Mac Studio 的性能总是开一些工程文件然后进行处理、渲染、导出之类的操作，看使用时间之类的。我对数码产品兴趣程度下降以后，现在只要不是不该卡顿的时候卡顿，或者处理时间长到让我皱眉，那就是「够用」。

　发热：我之前以为 Mac Studio 不会产生明显的热，实际上…… 真的没有。不搞点大动作基本就是金属的冷冰冰质感。有一次稍微温了是我在升级系统。但我发现即使不热，它的风扇也是在工作的，出来的风也凉凉的，可能是为了长时间使用更稳定吧。

　音频：它声音得从底部出来。可能是我摆放角度的问题，我的 Mac Studio 其实是后侧朝桌子右侧的（转了 90°），声学设计可能不是最优。外放听感上没有 MBP 好。目前我也不太用 Mac Studio 听歌。我喜欢在无声的环境中专注地使用电脑，外加保护听力，于是声学这块我也没啥不满意的。

　小问题：和 MacBook Pro 比起来，唯一我觉得不够丝滑的是解锁这块。笔记本有Touch ID，可以开盖，指纹解锁一气呵成。但是 Mac Studio 得先激活键盘和鼠标，等几秒钟屏幕亮起来了输密码解锁，这一块会有一些延迟。倒是有带 Touch ID 的 Magic Keyboard，但是我已经有自己的键盘了，也就没必要增加实体了。如果 Mac Studio 机身上集成一个指纹 / 掌纹解锁那就百分百丝滑了。

## 尾声

　有点期待 6 年后的电脑会是什么样。到时候 AI 本地模型可能已经大行其道了吧。又或者家里以后有个中心服务器，其他地方放个拇指大的「机顶盒」就能输出远程桌面的视频流了。又或者那时候电脑何必还是电脑的形态，可能机器狗也能当主机用了。

　在那些科幻场景到来之前，Mac Studio 将是我稳定的生产力工具。多多指教了。
